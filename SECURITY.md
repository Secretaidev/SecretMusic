# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| Latest  | :white_check_mark: |
| Older   | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability within SecretMusic, please report it to us:

- **Email:** secretfetcher@gmail.com
- **Telegram:** [@its_me_secret](https://t.me/its_me_secret)
- **Support Group:** [@song_assistant](https://t.me/song_assistant)

We take all security reports seriously and will work to resolve them as quickly as possible.

## Response Timeline

- **Acknowledgment:** Within 24 hours
- **Initial Assessment:** Within 48 hours
- **Resolution:** Based on severity
